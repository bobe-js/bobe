

1. 字符内自动转义 ✅

   ```pug
   div text={'123\n'}
   ```

   自动转义为

   ```pug
   div text={'123\\n'}
   ```

2. 静态复制支持计算 ✅

   ```pug
   ${Child} value=${(data) => `${data.value}元`}
   ```

3. Ref

   ```pug
   div ref={xxx}
   ```

4. Context

   ```pug
   ctx xxxStore={a}
   ```

5. 自定义 Effect，或者 Noop Effect
   

6. 组件支持对属性赋值在当前位置中赋值为 模板片段

   ```pug
   ${Child} header=
   	div text="header头"
   | 
   	div	text="children"
   ```
