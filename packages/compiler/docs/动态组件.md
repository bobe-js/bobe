## dynamicDeclaration

生成一个 dynamicNode

1. 首屏渲染

   1. component 节点

      1. createComponentData， 生成对应节点信息给 node
      2. 直接解析动态节点的所有属性
      3.  处理 tokenizer 转换（即切换到这个组件的上下文）

   2. text 节点

      创建一个 文本节点， 并赋值 props， 通过 effect 响应后续的变更

      ```tsx
      node = this.createNode('text');
      const str = valueIsMapKey ? value : this.getFn(data, value)();
      // 更新渲染如果是新字符串，会做重新赋值
      node.textContent = str;
      ```

2. 更新渲染

   1. 原来是 component 节点： 将原来 component 卸载
      1. 现在是另一个 component
      2. 现在是 text 节点，将文本节点插入在 anchor 之前
   2. 原来是 text 节点： 
      1. 现在是 component
         1. 卸载 text 节点
      2. 现在是新 text 
         设置 node.textContent = str;
